# Welcome to your Expo app 👋

DESCRIPCIÓN DEL PROYECTO

VetCare es una aplicación móvil desarrollada en React Native que permite a una clínica veterinaria gestionar de manera eficiente las solicitudes de atención de sus pacientes. La aplicación permite registrar, visualizar, editar y eliminar solicitudes, así como filtrarlas por estado y realizar búsquedas.
